Data Structures

1. Array

Basic static collection.

int arr[5] = {10, 20, 30, 40, 50};
// Access: arr[0] -> 10
// Update: arr[1] = 99

2. Stack (LIFO)

Last In, First Out.

stack<int> s;
s.push(10);
s.push(20);
s.pop(); // Removes 20
cout << s.top(); // 10

3. Queue (FIFO)

First In, First Out.

queue<int> q;
q.push(10);
q.push(20);
q.pop(); // Removes 10
cout << q.front(); // 20

4. Binary Tree

Nodes with max 2 children.

struct Node {
  int data;
  Node *left, *right;
  Node(int val) { 
    data = val; left=right=NULL; 
  }
};

6. Linked List

Dynamic nodes with pointers.

struct Node {
  int data;
  Node* next;
};
// Insert head
void push(Node** head, int val) {
  Node* n = new Node();
  n->data = val;
  n->next = (*head);
  (*head) = n;
}
